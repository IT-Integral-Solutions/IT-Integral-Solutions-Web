# IT Integral Solutions – Website

Landing page oficial del estudio **IT Integral Solutions**.

Este repositorio contiene el código de la primera versión del sitio web institucional, diseñado como una landing moderna, rápida y optimizada para SEO.

---

# Objetivo del proyecto

El sitio tiene como objetivo:

- presentar los servicios del estudio
- generar leads de potenciales clientes
- mostrar proyectos y capacidades técnicas
- servir como base para futuras expansiones del sitio

La arquitectura fue diseñada para ser **simple, mantenible y escalable**.
